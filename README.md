one-off analyses, rendered at [docs.jimmyhoughjr.net](https://docs.jimmyhoughjr.net)
alongside the platform docs.

**Looking for the platform?** The Roost playbook, getting-started guide, and
toolbelt (`new-app.sh`, `publish-route.sh`) moved to
[jhoughjr/roost](https://github.com/jhoughjr/roost).

Here: `bin/usage-report.py` (daily ledger generator, runs via launchd),
`data/usage-ledger.json`, and the report markdown.
